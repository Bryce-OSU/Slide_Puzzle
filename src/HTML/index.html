Background Color Changes the background color. This is the color of the background behind the image.
Columns Changes the number of columns. Increasing the number of columns will also increase the overall number of squares. A larger number will make a puzzle that will take longer to complete. Requires the puzzle to be shuffled to see changes.
Rows Changes the number of rows. Increasing the number of rows will also increase the overall number of squares. A larger number will make a puzzle that will take longer to complete. Requires the puzzle to be shuffled to see changes.
Display Numbers? Displays square numbers. Squares are numbered in order left to right, top to bottom, starting from the top-left square.
Number Size Changes the text size of the square numbers.
Number Color Changes the text color of the square numbers.
Display Outlines? Displays square outlines. Squares will be highlighted when they are in the correct position.
Outline Size Changes the outline thickness of the square outlines.
Outline Color Changes the outline color of the square outlines.
Invert controls? The arrow keys, "wasd", or "ijkl" can be used to side squares. Toggling this will invert controls. Check this if movement feels backwerds.
Show Hint? Allows the use of a hint. The hint will display the completed image. Press shift to toggle the hint on and off.
Radomize the location of the empty square? Randomizes the location of the empty square. Default is the bottom right square. Makes the puzzle more difficult. Requires the puzzle to be shuffled to see changes.
