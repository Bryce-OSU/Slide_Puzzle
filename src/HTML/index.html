Background Color Changes the background color. This is the color of the background behind the image.
Columns Changes the number of columns. Bigger number = more difficult.
Rows Changes the number of rows. Bigger number = more difficult.
Display Numbers? Displays square numbers. Squares are numbered in order left to right, top to bottom, starting from the top-left square.
Number Size Changes the text size of the square numbers.
Number Color Changes the text color of the square numbers.
Display Outlines? Displays square outlines. Squares will be highlighted when they are in the correct position.
Outline Size Changes the outline tickness of the square outlines.
Outline Color Changes the outline color of the square outlines.
Invert controls? Default is "w" and "i" = up, "s" and "k" = down, "a" and "j" = left, "d" and "l" = right. Toggling this will invert controls. Check this if movement feels backwerds.
Show Hint? Check this to display the completed image. Shortcut: hold shift to display image.
Radomize the location of the empty box? Randomizes the location of the empty square. Default is the bottom right square.
